1、权限修饰符

1.1 什么是权限修饰符？

权限修饰符用于控制成员的访问范围。可以修饰成员变量，方法，构造器，类，不同权限修饰符有不同的访问权限！

1.2 四种权限修饰符访问范围

从大到小：public > protected > 缺省（不写） > private

2、final 关键字

final 关键字
a. 修饰类：不能被继承
b. 修饰方法：能被继承，不能被重写
c . 修饰变量：常量，基本数据类型数值不能修改，引用类型地址值不能修改（只有一次赋值的机会）
